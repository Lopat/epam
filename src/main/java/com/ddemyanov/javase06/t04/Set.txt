~~~~~Set~~~~~
Типичные реализации - HashSet, TreeSet, LinkedHashSet.
Представляет собой набор значений, без индекса, аналог математического множества, благодаря чему на позволяет
хранить в коллекции дубликаты, т.е. в любой реализации каждый элемент хранится в единственном экземпляре.

~~~~~Использование~~~~~
HashSet хранит множество объектов, эффективно размещенных с помощью хеш-функции.
TreeSet хранит множество объектов в упорядоченном заданным компаратором виде.
LinkedHashSet хранит множество объектов в порядке их добавления в коллекцию.

~~~~~Методы~~~~~
Iterator iterator() - возвращает итератор коллекции.
int size() - возвращает размер коллекции.
boolean isEmpty() - проверка на пустую коллекцию.
boolean contains(Object o) - проверка содержит ли коллекция объект о.
boolean add(Object o) - добавляет элемент в коллекцию.
boolean addAll(Collection c) - добавляет все элементы из другой коллекции в текущую.
Object[] toArray() - преобразует множество в массив.
boolean remove(Object o) - удаляет объект о из коллекции.
boolean removeAll(Collection c) - удаляет все элементы другой коллекции из текущей.
boolean retainAll(Collection c) - (retain — сохранить). Выполняет операцию "пересечение множеств".
void clear() - очищает коллекцию.
Object clone() - клонирует коллекцию.
